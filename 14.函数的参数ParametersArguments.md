```java
//在function定义的时候里面的参数叫做parameters
public void fans_show_hands(String place){
    //歌手向不同地点的粉丝们打招呼
    System.out.println("Hello，来自" + place + "的朋友们！你们好！");
    
}

//当你在真正调用函数，并且pass进去参数的时候，这个“真实”的参数叫做arguments
fans_show_hands("中国");


```
