##### 变量以及常量

 

###### 1.变量需要声明：在Java中每一个变量（Variable）必须要有一个类型(Type)

```java
int days;
long population;
boolean man;
double income;
double number;
```

###### 2.变量声明之后需要初始化

```java
int days;
System.out.println(days);
```

ERROR-variable not initialized!

```java
可以这样初始化
int days;
days = 7;

当然也可以这样初始化并放在不同位置

int days =7;
System.out.println(days);
double incomes =100;
```
